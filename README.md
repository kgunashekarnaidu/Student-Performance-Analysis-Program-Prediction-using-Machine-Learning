#  Student Performance Analysis & Program Prediction

##  Project Overview

This project presents a comprehensive data analysis and machine learning approach to evaluate student performance and predict their academic program classification (Academic or Vocational). It demonstrates an end-to-end data science pipeline, including data preprocessing, exploratory analysis, model development, and performance evaluation.

---

##  Problem Statement

Educational institutions often require data-driven insights to understand student performance and guide academic decisions. This project aims to analyze key influencing factors and build a predictive model to classify students into appropriate academic programs.

---

##  Methodology

###  Data Preprocessing

* Managed missing values and ensured data consistency
* Encoded categorical variables into numerical formats
* Applied feature scaling techniques
* Identified and handled outliers

###  Exploratory Data Analysis (EDA)

* Performed univariate and bivariate analysis
* Visualized data distributions using histograms and box plots
* Analyzed relationships using scatter plots and correlation matrices

###  Model Development

* Implemented a **Linear Regression model**
* Trained the model using processed and scaled data

###  Model Evaluation

* Evaluated model performance using:

  * Mean Squared Error (MSE)
  * R² Score
* Assessed predictive capability and reliability
